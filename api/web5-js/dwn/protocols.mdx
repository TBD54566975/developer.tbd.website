### query(target, request)

#### Parameters

- `target` - 
- `request` - The query request must contain the following:

    - `author`: The decentralized identifier of the `did`` signing the query. This may be the same as the target parameter if the target and the signer of the query are the same entity, which is common for an app querying the DWeb Node of its own user.
    - `message`: The properties of the DWeb Node Message Descriptor that will be used to construct a valid DWeb Node message.
    - `filter`: an object that defines a set of properties to filter results.
    - `protocol`: a URI that represents the protocol being queried for.

#### Return Value

The response of the request as an object.

#### Code Example

```
const web5 = new Web5();
const response = await web5.dwn.protocols.query('did:example:alice', {
  author: 'did:example:alice',
  message: {
    filter: {
      protocol: "https://decentralized-music.org/protocol",
    }
  }
});
```
