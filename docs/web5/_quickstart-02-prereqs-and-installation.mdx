<p><a href="https://docs.npmjs.com/downloading-and-installing-node-js-and-npm"

<p>Ensure you have the Node Package Manager, <code>npm</code>, installed and on your system's <code>$PATH</code>.</p>

<h2>Installation</h2>

Commands below will be done in your terminal.

<h4>1. Create Directory:</h4>
<p>This will be the home of your new Web5 app.</p>
<pre>mkdir web5-app</pre>
<pre>cd web5-app</pre>

<h4>2. Install Web5:</h4>

<p>We'll be using the NPM package manager. Run <code>npm</code> to install Web5:</p>

<pre>npm install @tbd54566975/web5@0.4.0</pre>

:::info
After npm resolves the dependency you may see a few warnings which you can ignore for now. 
:::

This will add Web5 as a dependency of the project, which you should now see in <code>package.json</code>. It will also resovle its transitive dependencies in a file called <code>package-lock.json</code> and place those dependencies in a folder called <code>node_modules</code>.

To verify everything is working, open <code>package.json</code> or run in Mac/Linux systems:
<pre>cat package.json</pre>

This should look like:
```json
{
  "dependencies": {
    "@tbd54566975/web5": "^0.4.0"
  }
}
```
<h1>NOT STARTED EDITING BELOW THIS LINE</h1>

<h4>3. Import Dependencies into app</h4>
    <pre>mkdir src</pre>

    <pre>cd src</pre>

    `touch index.js`

    Open `web5-app/index.js` in your editor

    Paste at the top of index.js:
     `import { Web5 } from '@tbd54566975/web5'`


## 2. Import `Web5`

You'll need to import the `Web5` class for your JavaScript application to reference it.
