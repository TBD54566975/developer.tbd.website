# Web5 Quickstart Widget

This repo shows the Web5 Quickstart in action. To run the project, follow these steps:

1. Clone into your local workspace. Replace $yourUsernameOrOrg with your GitHub username or org.

```bash
$> git clone https://github.com/$yourUsernameOrOrg/developer.tbd.website.git
```

2. Navigate to the project directory:
```bash
$> cd developer.tbd.website/examples/web5-quickstart-widgets
```

3. Install project dependencies using pnpm:

```bash
$> pnpm install
```

4. Run the project

```bash
$> pnpm start
```

The project will be running on http://localhost:2024/
