# Admonition Component

<div className="">
<p>The `Admonition` component is used to display a message with a specific style and icon.</p>

## Properties

| Property    | Type              | Required | Default | Description                       |
| ----------- | ----------------- | -------- | ------- | --------------------------------- |
| `variant`   | `string`          | Yes      | `""`    | The variant of the admonition.    |
| `children`  | `React.ReactNode` | Yes      | `""`    | The content of the admonition.    |
| `className` | `string`          | No       | `""`    | The class name of the admonition. |

## Usage

```jsx title="Admonition Component"
<Admonition variant="info">This is an info message. <Admonition/>
```

{/* variant Variant = "info" | "warning" | "tip" | "danger" | "note"; */}
