# Hero Component

<div className="component-guide text-white">
<p>The `Hero` component is used to display a title, content, and a button, typically used for landing pages or promotional sections.</p>

## Properties

| Property     | Type     | Required | Default | Description                                                |
| ------------ | -------- | -------- | ------- | ---------------------------------------------------------- |
| `title`      | `string` | Yes      | `""`    | The main title of the Hero component.                      |
| `content`    | `string` | Yes      | `""`    | The content or description text displayed below the title. |
| `url`        | `string` | Yes      | `""`    | The URL that the button will link to.                      |
| `buttonText` | `string` | Yes      | `""`    | The text displayed on the button.                          |

## Usage

```jsx title="Hero Component"
<Background
  squareCount={5}
  bgColor="yellow"
  className="grid items-center justify-center"
  childrenClassName="grid  justify-center items-center"
>
  <HeroCard
    title="This is a title"
    buttonText="Text for the button"
    url="localhost:3000"
    content="Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua."
  />
</Background>
```
