# Icon Component

<div className="component-guide text-white">
<p>The `IconButton` component is used to display a button with an icon and text.</p>

## Properties

| Property   | Type              | Required | Default | Description                        |
| ---------- | ----------------- | -------- | ------- | ---------------------------------- |
| `children` | `React.ReactNode` | Yes      | `""`    | The icon to display on the button. |

## Usage

```jsx title="Icon Button Component"
<IconButton>
  <Swift />
</IconButton>
```
