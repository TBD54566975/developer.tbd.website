# ImportSrc Component

<div className="">
<p>The `ImportSrc` component displays the correct import statement for the `Web5` API based on the specified CDN source. It dynamically generates an import URL based on the selected CDN (`unpkg` or `jsdelivr`) and the `WEB5_VERSION` from the Docusaurus custom fields.</p>

## Properties

| Property | Type     | Required | Description                                                  |
| -------- | -------- | -------- | ------------------------------------------------------------ |
| `src`    | `string` | Yes      | Specifies the CDN source. Accepts `"unpkg"` or `"jsdelivr"`. |

## Usage

To use the `ImportSrc` component, specify the `src` prop to indicate the desired CDN source.
