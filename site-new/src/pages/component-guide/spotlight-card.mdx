## SpotLightCard Component

<div className="component-guide text-white">
<p>The SpotLightCard component is used to display a spotlight section, which includes an image, a title, a handle (typically a heading or name), and optional children (usually text or other elements). It can be used to highlight featured content or profiles.</p>

## Properties

| Property  | Type            | Required | Default  | Description                                                                       |
| --------- | --------------- | -------- | -------- | --------------------------------------------------------------------------------- |
| src       | string          | Yes      | ""       | The URL of the image displayed in the card.                                       |
| alt       | string          | No       | ""       | Alternative text for the image, used for accessibility.                           |
| className | string          | No       | ""       | Additional CSS classes to apply custom styling to the card.                       |
| tone      | ToneTypes       | No       | "yellow" | The tone or color for the title text, can be "teal", "purple", "yellow", etc.     |
| title     | string          | Yes      | ""       | The title text displayed above the heading.                                       |
| handle    | string          | Yes      | ""       | The heading or handle displayed in the card (usually a name or main heading).     |
| children  | React.ReactNode | No       | ""       | The content displayed below the handle, can be text or additional React elements. |

## Usage

```jsx
<SpotLightCard
  src={docusaurusSocialCard}
  alt="Example Image"
  title="Featured Title"
  handle="Featured Handle"
  tone="teal"
>
  This is the body text for the spotlight card.
</SpotLightCard>
```
