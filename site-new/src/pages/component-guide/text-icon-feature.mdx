## TextIconFeature Component

<div className="component-guide text-white">
  <p>
    The TextIconFeature component is used to display a feature section with an
    icon, a title, and body text. It is useful for showcasing various features
    or highlights with customizable colors and icons.
  </p>
</div>

## Properties

| Property | Type   | Required | Default  | Description                                                                               |
| -------- | ------ | -------- | -------- | ----------------------------------------------------------------------------------------- |
| body     | string | Yes      | ""       | The body text displayed in the feature section.                                           |
| title    | string | No       | ""       | The title text displayed above the body.                                                  |
| icon     | func   | No       | Heart    | A custom icon to display. Defaults to the Heart icon.                                     |
| variant  | string | No       | "yellow" | The text and icon color. Can be "teal", "purple", "yellow", "yellow-shade-1", or "black". |

## Usage

```jsx
<TextIconFeature
  title="Feature Title"
  body="This is the body of the feature section, describing the feature in detail."
  variant="teal"
/>
```
