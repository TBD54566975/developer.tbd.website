## Overview
Every modifying method of the [Protocols](protocols) class returns an instance of a `Protocol` object, which is a representation of the protocol(s) involved in the operation. [Learn more about Protocols](/docs/web5/learn/protocols).

## Methods

### send(did)

Sends the protocol to the remote DWN endpoints of a provided DID. This function only needs to be called if you'd like to send the protocol to a user's remote DWNs instantly and cannot wait for [sync](/docs/web5/learn/sync#sync-intervals) to occur.

#### Parameters

    {
      data: {
