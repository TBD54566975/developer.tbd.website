## Getting Started
Depending on the language you're programming in, you'll want to <LanguageSwitchLink text='link to different content' links='{"Kotlin": "https://kotlinlang.org/", "JavaScript": "https://www.javascript.com/"}'/>. Sometimes, you'll use the exact same link [for both languages](https://www.google.com/);

Regardless, you can switch programming language and plain text content using blocks like the one below.
<LanguageSwitchBlock>
    <div language="Kotlin">
    <p>Because sometimes you might want to write Kotlin code talking about Key Management:</p>
    <LanguageTabBar />
        <CodeBlock language="kotlin">
{` fun initKeyManagement(env: string, didUri: string){
    var keyManager: KeyManager;
\n
    // Determine which key manager you'd like to use based on
    // the environment
    when(env) {
        "development" -> keyManager = InMemoryKeyManager();
        "production" -> keyManager = AwsKeyManager()
    }
\n
    // Initialize or load a DID
    var did: DidDht;
    if (didUri == null) {
        // Create a new DID
        val did = DidDht.create(keyManager)
    } else {
        // Load existing DID
        val did = DidDht(uri = "some_uri", keyManager)
    }
}
`}
        </CodeBlock>
    </div> 
    <div language="JavaScript">
    <p>Because sometimes you might want to write JavaScript code talking about credential creation:</p>
    <LanguageTabBar />
        <CodeBlock language="js">
{`
const vc = await VerifiableCredential.create({
  type: 'SwiftiesFanClub',
  issuer: fanClubIssuerDid.did,
  subject: aliceDid.did,
  data: new SwiftiesFanClub('Stan', true)
});
`}
        </CodeBlock>
    </div> 
</LanguageSwitchBlock>

And both of these blocks should be able to live in a single document.
