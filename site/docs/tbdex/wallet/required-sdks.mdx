# Required SDKs

To implement and operate as a Wallet on a tbDEX network, youâ€™ll need to use the following SDKs:


* [**web5/dids**](https://www.npmjs.com/package/@web5/dids) - Enables the creation of [Decentralized Identifiers (DID)](/docs/web5/learn/decentralized-identifiers) for your customers and perform any other required cryptographic functions, you can use the DIDs package.
* [**web5/credentials**](https://www.npmjs.com/package/@web5/credentials) - Enable the presentation of [Verifiable Credentials (VC)](/docs/web5/learn/verifiable-credentials) to PFIs on your customers' behalf. You may also want to verify that the PFIs you wish to engage with have certain credentials.
* [**tbdex/client**](https://www.npmjs.com/package/@tbdex/http-client) - Enables your application to send tbDEX messages to PFIs.

## Install SDKs
To install `@web5/dids`:

```bash
npm i @web5/dids
```

To install `@web5/credentials`:

```bash
npm i @web5/credentials
```

To install `@tbdex/http-client`:

```bash
npm i @tbdex/http-client
```

## Import Classes

To send and receive tbDEX messages, import the following classes:

```js
import { TbdexHttpClient, Rfq, Quote, Order} from '@tbdex/http-client';
import { VerifiableCredential, PresentationExchange } from "@web5/credentials";
```
