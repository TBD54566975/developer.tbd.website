# Upgrade to DWA

A **Decentralized Web App (DWA)** is a Progressive Web App (PWA) that incorporates Web5 functionalities, providing offline capabilities, Decentralized Resource Locators (DRLs), and offline synchronization of Decentralized Web Nodes (DWNs). For a practical implementation, refer to the [DWA starter app](https://github.com/TBD54566975/dwa-starter).

## Import packages

```js
import { activatePolyfills } from '@web5/api';
```

## Set up a service worker
Configure a service worker to activate Web5 polyfills while managing caching and data synchronization tasks.

```js
// Activate Web5 Service Worker Polyfills
activatePolyfills({
  onCacheCheck() {
    return {
      ttl: 30000,
    };
  },
});
```
