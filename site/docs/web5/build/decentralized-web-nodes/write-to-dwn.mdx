# Write to a DWN

The following snippet allows you to create a record in a DWN:

```js
// Create a plain text record
const { record } = await web5.dwn.records.create({
    data: "Hello Web5",
    message: {
        dataFormat: 'text/plain',
    },
});
```

The `data` property can be a string, JSON, blob, stream, or file. For more information on writing records to DWNs, see our [API Guide](/api/web5-js/dwn/records#createrequest).
