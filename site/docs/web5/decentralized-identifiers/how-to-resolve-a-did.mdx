# Resolve a DID

Resolving a DID refers to the process of retrieving the [DID document](/docs/web5/decentralized-identifiers/did_documents) associated with a [Decentralized Identifier](/docs/web5/decentralized-identifiers/what-are-dids). The DID Document contains important information such as public keys, authentication methods, service endpoints, and more. 


## Retrieving the DID Document

When resolving a DID, make sure you are passing the [DID URI](/docs/web5/decentralized-identifiers/how-to-create-did#diddht) to the resolver function.
